# Calculadora App

## Descrição

Esta calculadora feita no MIT app Inventor, possui as operações basicas e também, resto de divisão, potência e raiz quadrada.

## Como usar

1. Insira os valores que deseja calcular nos campos "Valor 1" e "Valor 2".
2. Selecione a operação desejada sendo possivel a escolha de: soma, subtração, multiplicação, divisão, resto de divisão, potência ou raiz quadrada.
3. Clique no botão "Calcular" para obter o resultado.

## Recursos

- Operações matemáticas padrões: soma, subtração, multiplicação e divisão.
- Operações Extras: 
  - Resto de divisão: retorna o resto de uma divisão. (Sendo o Valor que você deseja retornar, no campo "valor 1")
  - Potência: eleva um número a uma potência especificada. (Sendo o valor que você deseja elevar, no campo "Valor 1")
  - Raiz quadrada: calcula a raiz quadrada de um número. (Sendo o valor que você deseja calcular, no ocampo "Valor 2")
- Interface instintiva e comun de se usar.
